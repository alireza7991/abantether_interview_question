**An implementation of AbanTether Interview Question.**

_By Alireza Forouzandeh Nezhad_

### Prerequisites
- Docker
- Justfile

## Commands

### 1- Build docker image
```
just build
```
### 2- Run/Stop the application:
```
just run
```
or to run in background:
```
just run-detached
```
and to stop it:
```
just stop
```
### 3- Running Tests
```
just test
```
